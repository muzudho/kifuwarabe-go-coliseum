# Run app on Windows

## Start

```shell
# 引数には、このアプリ、白番のエンジン、黒番のエンジンの作業ディレクトリを指定してください
kifuwarabe-go-coliseum ^
--workdir C:/Users/むずでょ/go/src/github.com/muzudho/kifuwarabe-go-coliseum/workspace ^
--workdirw C:/Users/むずでょ/go/src/github.com/muzudho/kifuwarabe-go-coliseum/workspace-w ^
--workdirb C:/Users/むずでょ/go/src/github.com/muzudho/kifuwarabe-go-coliseum/workspace-b
```
